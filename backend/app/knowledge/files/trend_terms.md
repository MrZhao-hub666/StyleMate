# 时尚趋势关键词库

用于穿搭描述和 AI 生图 prompt 优化。

## 一、流行风格

### 当代主流风格
- **Clean Fit**：干净利落的基本款搭配，强调剪裁和质感，中性色调为主
- **Old Money / 老钱风**：低调奢华，羊绒、真丝、精纺羊毛，驼色/藏青/米白为主
- **Y2K**：千禧年复古，低腰裤、露脐装、亮色、金属感
- **Gorpcore**：户外机能风，冲锋衣、工装裤、登山鞋融入日常
- **City Boy**：日系宽松少年感，Oversize衬衫+宽松裤
- **Vibe / 氛围感穿搭**：强调整体氛围而非单品，柔和色调
- **Balletcore**：芭蕾风，芭蕾平底鞋、裹身针织、粉色系、纱裙
- **Dopamine Dressing / 多巴胺穿搭**：高饱和度亮色、撞色搭配、愉悦感

### 经典风格
- **极简风(Minimalist)**：少即是多，基础款+中性色调，不超3色
- **法式风(French Chic)**：条纹衫、贝雷帽、高腰牛仔裤、慵懒感
- **日系风(Japanese Casual)**：宽松、叠穿、棉麻质感、低饱和度
- **韩系风(K-style)**：修身+宽松搭配、浅色系、注重比例
- **街头风(Streetwear)**：卫衣、运动鞋、Logo、oversize
- **学院风(Preppy)**：Polo衫、格纹、卡其裤、乐福鞋
- **复古风(Vintage)**：古着单品、复古配色、年代感元素
- **性冷淡风(Normcore)**：看不出品牌的"普通"穿法，实则精挑细选

## 二、场景关键词（用于生图 prompt）

### 肖像写真场景
- 日系校园风：樱花树下、JK制服、柔光、清新
- 港风复古：霓虹灯、红唇、大波浪、胶片感
- 赛博朋克：霓虹灯、金属反光、暗色调、未来感
- 古风汉服：水墨画背景、飘带、竹林、画意
- 职业证件照：纯色背景、正面、自然光、简洁
- 韩式证件照：柔光、浅灰背景、自然微笑

### 穿搭街拍场景
- 都市通勤：写字楼背景、咖啡杯、利落干练、自然走动
- 周末探店：咖啡厅内、柔和灯光、慵懒氛围、侧拍
- 度假旅行：海滩/山景、草帽、飘逸长裙、逆光
- 夜景街拍：城市霓虹、夜灯、风衣、动态模糊
- T台秀场：红毯、聚光灯、气势、正面全身

## 三、AI 生图 Prompt 优化技巧
1. 加入构图描述："全身照"、"半身肖像"、"正面站立"、"侧身45度"
2. 加入光线描述："自然光"、"黄金时刻"、"影棚光"、"柔光"
3. 加入风格参考："杂志大片"、"街拍风格"、"电影镜头感"
4. 加入细节："高清"、"细腻的纹理"、"自然的褶皱"
5. 避免的词汇：避免过于复杂的描述，保持简洁明确
