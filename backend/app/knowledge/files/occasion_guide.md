# 场合着装规范

## 一、面试着装

### 商务正装（传统行业：金融/法律/咨询）
- 男士：深色西装（藏青/深灰）、白色或浅蓝衬衫、素色领带、黑色皮鞋
- 女士：西装套装或西装裙、浅色衬衫、中跟皮鞋、简约配饰
- 配色：不超过3色，以深色+白为主
- 禁忌：运动鞋、牛仔裤、过于鲜艳的颜色、过多首饰

### 商务休闲（互联网/创业公司）
- 男士：衬衫（可不打领带）、休闲西装或针织衫、深色长裤、皮鞋或商务休闲鞋
- 女士：衬衫/针织衫+长裤/及膝裙、低跟鞋
- 配色：可略活泼，但保持专业感
- 核心：整洁、合身、不过于随意

## 二、约会着装

### 初次约会
- 女士：连衣裙或衬衫+半裙，温柔色系（粉色、米色、浅蓝），体现女性气质
- 男士：衬衫+休闲裤或修身牛仔裤，干净清爽
- 整体：得体大方，不过分正式也不过分随意

### 日常约会
- 根据场合决定：咖啡厅→休闲精致、餐厅→稍正式、户外→运动休闲
- 核心：展现个人风格，保持干净整洁

### 配色建议
- 推荐：柔和色调（粉、米白、浅蓝、浅灰）
- 避免：全黑（过于严肃）、荧光色（过于扎眼）

## 三、商务/职场日常

- 保持专业形象，颜色以中性色为主（黑、白、灰、蓝、驼）
- 版型合身，面料挺括
- 可用丝巾/手表等配饰增加精致感
- 避免：拖鞋、运动短裤、过于暴露、过于紧身

## 四、休闲日常

- 自由发挥，以舒适为前提
- 基础款T恤+牛仔裤/休闲裤为经典组合
- 可加入个性单品（印花T恤、亮色鞋、帽子）
- 注意：保持基本配色协调即可

## 五、运动健身

- 功能性优先：吸湿排汗面料、弹性面料
- 合身但不紧绷
- 运动鞋根据运动类型选择（跑步/训练/球类）
- 配色可大胆鲜艳

## 六、晚宴/正式社交

- 女士：晚礼服/鸡尾酒裙、高跟鞋、精致手包
- 男士：深色西装/燕尾服、礼服衬衫、领结
- 深色为主，可用金属色/亮片点缀

## 七、场合-颜色对照

| 场合 | 推荐颜色 | 避免颜色 |
|------|---------|---------|
| 面试 | 藏青、深灰、白、浅蓝 | 红、荧光色、花哨图案 |
| 约会 | 柔粉、米白、浅蓝、浅灰 | 全黑、过于花哨 |
| 商务日常 | 黑、白、灰、蓝、驼 | 荧光色、过于鲜艳 |
| 休闲日常 | 无限制 | 无 |
| 运动 | 任意鲜艳色 | 无 |
| 晚宴 | 黑、深蓝、酒红、金银 | 过于休闲的颜色 |
