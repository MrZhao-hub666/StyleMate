# 穿搭黄金法则

## 一、比例法则

### 上下身比例
- **黄金比例 3:7 或 4:6**：通过高腰设计或塞衣角提升腰线
- 上短下长显腿长
- 上长下短适合高个子

### 版型搭配原则
- **上紧下松**：修身T恤 + 阔腿裤/伞裙
- **上松下紧**：宽松卫衣 + 紧身裤/铅笔裤
- **全身同松**：Oversize风格，需搭配露肤（露脚踝/手腕）
- **全身同紧**：适合身材较好者，注意场合

### 长度搭配
- 短上衣 + 高腰下装：优化比例
- 长外套 + 短内搭：层次感
- 中长外套 + 及膝裙/九分裤：稳重优雅

## 二、层次叠穿

### 基础叠穿公式
1. **衬衫 + 毛衣**：衬衫领子和下摆露出
2. **T恤 + 衬衫（敞开）**：休闲街头感
3. **高领打底 + 衬衫 + 外套**：冬季层次感
4. **连衣裙 + 外搭（开衫/西装）**：通勤优雅

### 叠穿要点
- 内层薄、外层厚
- 颜色由浅到深，或由深到浅渐变
- 不同材质搭配更有层次（针织+皮质+棉质）

## 三、配饰运用

### 基本配饰
- 手表：提升质感，商务选皮质表带，休闲选金属/帆布
- 腰带：分割上下身比例的关键
- 包包：颜色建议与鞋子呼应
- 帽子：棒球帽（休闲）、贝雷帽（文艺）、礼帽（优雅）

### 配饰原则
- 全身亮点不超过2处
- 配饰颜色与服装中某一颜色呼应
- 金属色（金银）可搭配大多数颜色

## 四、场合适配法则

| 法则 | 说明 |
|------|------|
| **三色原则** | 全身不超过3个主色调 |
| **TPO原则** | Time(时间) + Place(地点) + Occasion(场合) 三位一体 |
| **露肤平衡** | 上露下不露，前露后不露（正式场合） |
| **面料协调** | 全身面料厚薄一致（夏不配冬） |

## 五、常见搭配错误

1. **全身logo/印花**：选择一处作为亮点即可
2. **运动鞋+正装**：除商务休闲外不推荐
3. **袜子的颜色**：深色鞋配深色袜，切勿白袜+黑皮鞋
4. **过于紧身/过于宽松**：根据身材选版型
5. **颜色过多**：超过4色显杂乱
