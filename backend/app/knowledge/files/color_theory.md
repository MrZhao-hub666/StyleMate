# 色彩搭配理论

## 一、色彩基础

### 色相环
- 12色相环：红、橙红、橙、黄橙、黄、黄绿、绿、蓝绿、蓝、蓝紫、紫、紫红

### 色彩三要素
- 色相(Hue)：颜色的基本属性
- 明度(Value)：颜色的明暗程度
- 饱和度(Saturation)：颜色的鲜艳程度

## 二、配色法则

### 经典配色方案
1. **互补色搭配**：色相环上相对的颜色（如蓝+橙、红+绿）。视觉冲击力强，适合个性穿搭。
2. **邻近色搭配**：色相环上相邻的颜色（如蓝+蓝紫+紫）。和谐统一，不易出错。
3. **三角色搭配**：色相环上等距三角形的三种颜色。活泼丰富，需注意比例分配。
4. **单色搭配**：同一色相的不同明度/饱和度组合。极简高级，适合正式场合。

### 三色原则
- 全身搭配不超过3个主色调（不含黑白灰）
- 主色占60%、辅色占30%、点缀色占10%

### 黑白灰法则
- 黑、白、灰为万能过渡色，可与任何颜色搭配
- 不确定配色时，用黑白灰做主色调最安全

## 三、肤色与配色

### 暖色调肤色
- 适合：暖红、橙色、金色、驼色、军绿、米色
- 避免：冷蓝、蓝紫、荧光色

### 冷色调肤色
- 适合：蓝色、紫色、粉色、灰色、纯白
- 避免：橙色、土黄、暖棕

### 中性肤色
- 大多数颜色均适合，需注意明度协调

## 四、季节与配色

### 春季
- 推荐：浅粉、鹅黄、草绿、天蓝、米白
- 风格：清新明亮

### 夏季
- 推荐：白色、浅蓝、薄荷绿、珊瑚色
- 风格：清爽干净

### 秋季
- 推荐：驼色、酒红、橄榄绿、深棕、姜黄
- 风格：温暖质感

### 冬季
- 推荐：黑色、深灰、海军蓝、酒红、纯白
- 风格：沉稳厚重

## 五、常见配色方案速查

| 主色 | 推荐搭配色 | 风格 |
|------|-----------|------|
| 白色 | 任意颜色 | 清爽百搭 |
| 黑色 | 任意颜色 | 显瘦酷感 |
| 蓝色 | 白色、灰色、卡其 | 稳重斯文 |
| 红色 | 黑色、白色、牛仔蓝 | 吸睛醒目 |
| 绿色 | 白色、米色、棕色 | 自然活力 |
| 卡其 | 白色、黑色、蓝色 | 知性优雅 |
| 灰色 | 白色、粉色、蓝色 | 高级冷淡 |
